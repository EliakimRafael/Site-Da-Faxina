Política de Privacidade

A Limpeza Prime respeita a privacidade dos seus clientes e compromete-se a proteger os dados pessoais fornecidos através deste site.

1. Responsável pelo tratamento dos dados

O responsável pelo tratamento dos dados é Eliakim Rafael Dos Santos, representante da Limpeza Prime.

2. Dados recolhidos

Podemos recolher dados como nome, telefone, tipo de serviço solicitado e mensagem enviada através do formulário de orçamento.

3. Finalidade dos dados

Os dados são utilizados apenas para responder a pedidos de orçamento, contactar o cliente e prestar os serviços de limpeza solicitados.

4. Partilha de dados

A Limpeza Prime não vende, aluga ou partilha os dados pessoais dos clientes com terceiros, exceto quando necessário para cumprir obrigações legais.

5. Direitos do utilizador

O utilizador pode solicitar o acesso, correção ou eliminação dos seus dados pessoais entrando em contacto com a Limpeza Prime.

6. Contacto

Para questões relacionadas com privacidade, contacte a Limpeza Prime através do WhatsApp disponível no site.
